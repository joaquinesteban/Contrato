Términos & Condiciones

Acuerdo de Prestación de Servicios

Desarrollo Web · Automatizaciones · San Luis, Argentina

1. Acuerdo

Este acuerdo define los términos entre el desarrollador (en adelante, "el Freelancer") y el cliente (en adelante, "el Cliente") para la creación de una página web y/o la implementación de sistemas de automatización. El objetivo es garantizar entregables claros, tiempos definidos y una comunicación eficiente durante todo el proyecto.

2. Servicios a proporcionar

  • Desarrollo de página web profesional
  • Configuración de automatizaciones
  • Integración de APIs y herramientas
  • Configuración de base de datos
  • Optimización del sistema

3. Entregables y plazos

El Freelancer se compromete a entregar los avances según el cronograma acordado con el Cliente. Cualquier cambio de alcance que impacte en los plazos deberá ser acordado por escrito entre ambas partes antes de continuar el desarrollo.

4. Honorarios y forma de pago

El Cliente abonará al Freelancer el monto acordado de [monto] según el siguiente esquema: 50% al inicio del proyecto como seña, y el 50% restante contra la entrega final. Los pagos se realizarán mediante transferencia bancaria u otro medio acordado entre las partes.

5. Revisiones

El proyecto incluye hasta dos (2) rondas de revisiones sobre lo entregado. Cambios adicionales fuera del alcance original podrán cotizarse por separado.

6. Propiedad intelectual

Los derechos sobre el desarrollo final se transfieren al Cliente una vez efectuado el pago total acordado. El Freelancer se reserva el derecho de exhibir el proyecto como parte de su portfolio, salvo acuerdo de confidencialidad expreso.

7. Confidencialidad

Ambas partes se comprometen a mantener la confidencialidad de la información comercial, técnica o estratégica compartida durante el desarrollo del proyecto.

8. Cancelación

En caso de cancelación del proyecto por parte del Cliente, la seña inicial no será reembolsable y se abonará de forma proporcional el trabajo ya realizado hasta el momento de la cancelación.

9. Nada se mueve sin firma

El desarrollo del proyecto comienza únicamente una vez que ambas partes hayan firmado este acuerdo. Esto protege tanto al Freelancer como al Cliente.
